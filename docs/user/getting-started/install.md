# Install

## Binary

### Download from GitHub Releases

Download `smithy-<os>-<arch>` from the
[latest release](https://github.com/iorubs/smithy-cli/releases/latest).

### go install

```sh
go install github.com/iorubs/smithy-cli/cmd/smithy@latest
```

### Download from GitHub Releases

Download `agentsmithy-<os>-<arch>` from the
[latest release](https://github.com/iorubs/agentsmithy/releases/latest).

## Next steps

Next you'll need a `smithy-stack.yaml` config. See the
[Use Cases](./use-cases/simple-stack.md) section to find a scenario
that fits your needs, then follow the tip at the bottom to generate
your config with your agent.
