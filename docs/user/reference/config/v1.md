## MCP

MCP is a single MCP server entry. The referenced config is loaded by
mcpsmithy and served in-process by the supervisor.

| Field | Type | Required | Default | Exclusivity | Description |
|-------|------|----------|---------|-------------|-------------|
| `config` | string | yes | — |  | Path to the mcpsmithy config file (.mcpsmithy.yaml). Resolved<br />relative to the stack file's directory. |
| `transport` | [Transport](#transport) | no | — |  | Wire transport exposed by the server. Defaults to whatever the<br />referenced mcpsmithy config selects when unset. |
| `addr` | string | no | — |  | Listen address (host:port) for http/sse transports. Mirrors<br />mcpsmithy's --addr flag. Ignored for stdio. |
| `autorestart` | boolean | no | — |  | Restart on non-zero exit. Defaults to true when unset; set false<br />to opt out per-entry (one-shot or hand-managed servers). |

## Transport

Transport selects the wire protocol an MCP server exposes.

| Value | Description |
|-------|-------------|
| `stdio` | TransportStdio runs the server over stdio. |
| `http` | TransportHTTP serves the server over streamable HTTP. |
| `sse` | TransportSSE serves the server over Server-Sent Events. |
