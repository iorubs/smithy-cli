## Mode

Mode selects how a compose entry is launched.

| Value | Description |
|-------|-------------|
| `subprocess` | ModeSubprocess spawns the entry as a child process from `bin:`. |
| `goroutine` | ModeGoroutine runs the entry in-process via the embedded runtime. |


---

## Transport

Transport selects the wire protocol exposed by an agent.

| Value | Description |
|-------|-------------|
| `http` | TransportHTTP serves the agent over A2A HTTP. |
| `stdio` | TransportStdio runs the agent over stdio (chat-only, not supervised). |
